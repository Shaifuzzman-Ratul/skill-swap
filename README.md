# Skill Swap

Live Site: [https://your-client-live-url.com]( https://skillswap9.netlify.app/)

## Project Overview
SkillSwap is an interactive platform that allows users to offer, learn, and trade skills within their local area. Users can browse skill listings, rate experiences, and connect with local skill providers. The platform is responsive, features smooth animations, and provides a seamless single-page application experience.


## Features
- **User Authentication:** Login & Registration using Email/Password and Google OAuth  
- **Protected Routes:** Private pages accessible only to authenticated users  
- **Home Page:** Hero slider with popular skills displayed in a responsive grid  
- **Skill Listings:** Browse all skills with categories, ratings, price, and slots  
- **Skill Details Page:** View full skill information and book sessions  
- **Booking Functionality:** Book a session with a simple form and success toast  
- **My Profile Page:** View and update user info (Name, Email, Image)  
- **Responsive Design:** Mobile, tablet, and desktop optimized  
- **Smooth Animations:** Framer Motion, AOS, Swiper.js enhance UX  
- **Client-Side Validation:** Password validation, form validation, and feedback  
- **Dynamic Behavior:** SPA behavior with no errors on route reloads
